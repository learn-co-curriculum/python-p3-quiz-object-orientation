# OO Syntax

Consider the following code:

```python
class Pet:
  def __init__ (self, name):
    self.name = name

rabbit = Pet("Jabby")
rabbit.name = "Chokola"
print(rabbit.name)
```

After running the code in the example above, what
will`print(rabbit.name)`return?

## Correct

Chokola

> Yes! The rabbit's name was reassigned to "Chokola".
